# ğŸ”‘ Python Password Generator

This repository is a command-line tool that generates strong, random passwords of a user-specified length. The project includes two separate scripts with console outputs fully in Turkish (TR) and English (EN).

## Project Contents

There are two main Python scripts in this repository:

1. `TR_Password_Generator.py`: Generates passwords with a Turkish user interface and outputs.  
2. `EN_Password_Generator.py`: Generates passwords with an English user interface and outputs.

## Setup and Execution

To run this project, you need Python 3 and built-in Python libraries (`random`, `string`).

### 1. Cloning the Repository

Clone the repository (or clone it using GitHub Desktop):

```bash
git clone https://github.com/Alimerih0/Password_Generator.git
cd Password_Generator
```

### 2. Execution Commands

| Language | Command |
|-----------|----------|
| Turkish | `python TR_Password_Generator.py` |
| English | `python EN_Password_Generator.py` |
